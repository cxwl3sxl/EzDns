Features

功能特性

全功能 DNS 规则管理平台，满足从开发到生产的各种需求。

DNS 规则管理

通过 Web UI 或 REST API 轻松增删改查 DNS 解析规则。支持 A、AAAA、MX、NS、TXT 等多种记录类型。

智能规则引擎

支持固定 IP 和自动 IP 生成两种模式，基于优先级排序匹配，灵活应对各种 DNS 解析场景。

通配符域名匹配

支持 *.example.com 通配符格式，自动从子域名提取数字生成 IP，批量管理子域名解析。

JWT 身份认证

内置 JWT 登录认证，Token 持久化存储，自动续期。管理界面需认证后方可访问，保障安全。

亮 / 暗主题切换

跟随系统偏好自动切换或手动选择主题，在暗色与亮色模式间自由切换，适配不同使用环境。

跨平台服务部署

支持安装为 Windows 服务或 Linux systemd 服务。提供 install / uninstall / start / stop / status 命令。

CI/CD 自动构建

GitHub Actions 自动发布 Windows (win-x64) 和 Linux (linux-x64) 自包含单文件版本，无需安装运行时。

搜索与分页

内置域名搜索和分页功能，快速查找管理大量 DNS 规则，流畅管理数百条规则。

Architecture

项目架构

基于 .NET 8 + Vue 3 的现代化三层架构，清晰分离关注点。

EzDns.WebApi

ASP.NET Core 8 Web API & DNS 服务器

  • AuthController — JWT 登录认证
  • RulesController — 规则增删改查
  • EzDnsHostedService — DNS 后台服务
  • ServiceCommands — 服务安装/启停

EzDns.Core

共享模型与 DNS 解析逻辑

  • CustomRuleResolver — 规则解析引擎
  • RuleSort — 优先级排序
  • DnsRule — 规则数据模型
  • IRuleRepository — 仓储接口

EzDns.Web

Vue 3 + TypeScript SPA 前端

  • LoginView — 登录页面
  • RulesView — 规则管理页面
  • useTheme — 主题切换
  • useAuth — 认证状态管理

Get Started

快速开始

几分钟内即可启动完整的 DNS 管理服务。

1

启动后端 API

进入 API 项目目录，使用 dotnet CLI 启动。API 默认监听 http://0.0.0.0:8080，DNS 服务器同时启动。

> cd EzDns.WebApi
> dotnet run
2

启动前端（开发模式）

另一个终端启动 Vite 开发服务器，支持热更新。

> cd EzDns.Web
> npm install
> npm run dev
3

构建前端并集成

生产环境将前端构建产物复制到 API 的 wwwroot 目录，一次部署即可访问完整页面。

> cd EzDns.Web
> npm install
> npm run build
> mkdir -p ../EzDns.WebApi/wwwroot
> cp -r dist/* ../EzDns.WebApi/wwwroot/
4

登录使用

打开浏览器访问 API 地址，使用以下凭据登录管理界面：

用户名
admin
密码
admin123

⚠ 生产环境请务必修改默认密码和 JWT Secret
